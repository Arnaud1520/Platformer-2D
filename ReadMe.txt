Projet: Escape Island

Ce jeu de Plateforme créé dans le cadre d'un projet académique à Efrei Paris (2022).

Il a été fait à partie de la librairie graphique SFML en C++.

Pour lancer le jeu il suffit d'executer le fichier sfml-app dans le dossier src.
